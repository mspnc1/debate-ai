Frequently Asked Questions

Getting Started

What is Symposium AI?

Symposium AI is a revolutionary mobile app that lets you watch AI models debate any topic in real-time, chat with multiple AI providers simultaneously, and use your existing API keys to save money compared to multiple AI subscriptions.

How do I get started?

Simply download the app, add your API keys from providers like OpenAI, Anthropic, or Google, and you're ready to start chatting or watching AI debates! No subscription required if you bring your own API keys.

Do I need API keys to use the app?

Yes, you'll need at least one API key from supported providers (OpenAI, Anthropic, Google, etc.) to use the app. This "Bring Your Own Key" approach saves you money compared to paying for multiple AI subscriptions.

Features

What is the AI Debate Arena?

The AI Debate Arena is our signature feature where different AI models debate topics in real-time. You can watch AIs present arguments, counter-arguments, and reach conclusions on any topic. Premium users can create custom debate topics.

What is the Compare feature?

Compare lets you ask the same question to multiple AIs simultaneously and see their responses side-by-side. It's perfect for getting diverse perspectives or fact-checking information.

What is Hallucination Shield?

Hallucination Shield is our accuracy feature where multiple AIs fact-check each other's responses in real-time, helping reduce misinformation and AI "hallucinations."

Can I use different AI personalities?

Yes! Premium users can choose from 12 different personalities for each AI, including Professional, Friendly, Socratic, Creative, and more. This adds variety and makes conversations more engaging.

API Keys & Security

How do I get API keys?

You can get API keys directly from the AI providers:

  • OpenAI (ChatGPT): platform.openai.com
  • Anthropic (Claude): console.anthropic.com
  • Google (Gemini): makersuite.google.com
Each provider offers their own pricing, typically charging per token or message.

Are my API keys safe?

Yes! Your API keys are stored securely on your device only. They are never sent to our servers. All API calls are made directly from your device to the AI providers.

How much does it cost to use my API keys?

Costs vary by provider and usage. Most providers charge fractions of a cent per message. The app shows estimated costs for each model in Expert Mode (Premium feature).

Troubleshooting

Why am I getting API errors?

Common causes include:

  • Invalid or expired API key
  • Exceeded API rate limits or quota
  • No credit balance with the provider
  • Network connectivity issues
Try testing your API key in the Settings screen to diagnose the issue.

Why are responses slow?

Response speed depends on the AI model, your internet connection, and the provider's current load. You can enable streaming in Settings to see responses as they're generated rather than waiting for the complete response.

How do I report a bug?

Go to Help & Support in the app and tap "Contact Support" to send us an email with device information automatically included. We typically respond within 24-48 hours.
